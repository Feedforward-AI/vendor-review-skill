# Learn - Transferable Knowledge
Evaluate the LEARN (Transferable Knowledge) criterion for this vendor.

Assess whether using this tool builds generalizable AI capabilities:
- Does working with this tool help users understand broad AI principles—prompting, context management, model selection, evaluation?
- Or does it only teach users how to operate this specific tool's interface and workflow?
- Do users come away with skills portable to other AI tools and internal development?

Score as:
- "yes": Tool builds transferable AI skills and understanding
- "partially": Some learning value but limited to specific features
- "no": Only teaches how to use this specific tool
- "insufficient": Cannot assess due to lack of information

Provide:
1\. A 2-4 sentence assessment with specific evidence
2\. Trade-off statement (what org gains vs gives up)
3\. 2-3 specific questions to ask the vendor if gaps exist
